## 🎯 玩法说明

### 游戏系统

**属性系统：**
- 📊 **声望 (Rep)** - 决定朝堂影响力
- 💰 **银两 (Gold)** - 用于贿赂、装备、军粮
- ❤️ **生命 (HP)** - 归零即游戏结束

**势力关系：**
- 🏛️ **帝心** - 皇帝的好感度
- 🦊 **严党** - 严嵩派系的好感度
- 🌊 **清流** - 徐阶等清派的好感度
- 🏮 **内廷** - 司礼监的好感度
- ⚔️ **军方** - 戚家军等的好感度
- 👥 **民心** - 百姓的认可度

**判定系统：**
- 政治判定：影响官场策略成功率
- 武力判定：影响战斗和威慑
- 智谋判定：影响计谋和布局
- 清廉判定：影响道德抉择
- 魅力判定：影响交际和说服
